How to Use This Tool

Basic Instructions

  1. Load existing data (if any) using the "Load Data" button
  2. Add a new location by clicking "Add New Location" and filling out the form
  3. Set coordinates by clicking on the map
  4. Add photos to the location using the photo form and "Add Photo" button
  5. Save the location with the "Save Location" button
  6. Generate the JSON and copy it to your locations.json file

Important Notes

  • Location IDs should be unique and use lowercase letters with dashes (e.g., "chicago-il")
  • Photo paths should be relative to your website root (e.g., "portfolio-photos/example.jpg")
  • Alt text should follow your format: "Bird name (Scientific name) | Location"
  • You can edit existing locations by selecting them from the locations panel
  • Don't forget to copy the final JSON output to your locations.json file
