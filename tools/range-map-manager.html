Range Map Manager

How to Use This Tool

  • The left panel shows all bird species from your bird photo collection
  • A ● green dot means a range map file exists for this species
  • A ● red dot means no range map file exists yet
  • To add a range map, select a species and either:
    • Drag and drop an image file onto the drop area
    • Click "Browse" to select a file from your computer
    • Enter a URL to download an image
  • All range maps will be saved to the data/range-maps/ directory with filenames based on the scientific name
  • Click "Save Changes" when you're done to update the mapping file
